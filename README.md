# Second Order Meta Analysis
This repository contains code, functions, and worked examples accompanying _Methods for Second Order Meta Analysis_, a paper currently under review, authored by Larry V. Hedges, Gracie Hayes, and Paul Witte. The repository is intended to help researchers apply the proposed methods to their own data and reproduce the analyses presented in the manuscript.

# Repository Contents
## `\R`

This folder contains the core `R` scripts implementing the novel functions introduced in the paper. Users may download these scripts to:

- `functions.R` contains novel functions for performing fixed and random second order meta analysis.
- `examples.R` contains code for running the simple example of a second order meta analysis consisting of only 2 meta analyses.
- `application.R` contains code for the application to real data included in the paper.

Downloading the `\R` folder will allow the user to load the novel functions in order to perform second order meta analysis on their own data, or reproduce the results of the example and application.

## `Example.qmd` and `Example.html`

These files walk through the example generated in the `example.R` script with more detailed comments, graphics, and direction for the user to reproduce the results on their own. This example is a second order meta analysis consisting of only 2 meta analyses, first with no overlap and then with an overlapping effect size.

## `Application.qmd` and `Application.html`

These files walk through the application included in the paper. The full code for this application is found in `application.R`.
